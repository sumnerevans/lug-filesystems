## Authors/Presenters
- [Sumner Evans](https://github.com/sumnerevans)
- [Sam Sartor](https://github.com/samsartor)

## Topics Covered
- Background and History of Filesystems
- Responsibilities of Filesystems
- Linux Filesystems
- Windows Filesystems
- macOS Filesystems
- Flashdrive Filesystems
- Alternative Filesystems
- Network Filesystems
- Virtual Filesystems
